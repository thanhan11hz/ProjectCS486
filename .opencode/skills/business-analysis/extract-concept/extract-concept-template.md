## Review Checklist

### Explicit Concepts

* [ ] All concepts are directly supported by requirement text.
* [ ] No attributes are included as concepts.
* [ ] No implementation concepts are introduced.

### Normalization

* [ ] Every alias has a canonical concept.
* [ ] No semantic merging was performed without evidence.

### Categorization

* [ ] Every concept appears in exactly one category.
* [ ] Category assignment follows priority rules.

### Implicit Concepts

* [ ] Every implicit concept satisfies all extraction criteria.
* [ ] Supporting evidence is provided.

### Ambiguities

* [ ] Ambiguities are documented but not resolved.
* [ ] Each ambiguity is supported by requirement evidence.
